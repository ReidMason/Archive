# Skippythesnake's Components

## Developing
To develop pull down the repository and install dependencies.
```bash
npm install
```
This project uses [Storybook.js](https://storybook.js.org/) to test and develop components. So start the storybook server to preview development.
```bash
npm run storybook
```

## Building
To build the project just run the build command and push to the npm repository.
```bash
npm run build
npm publish
```

## Sources
- Created using [HarveyD/react-component-library](https://github.com/HarveyD/react-component-library) skeleton
- Makes use of [Headlesui](https://headlessui.dev/)
- General styling using [Tailwindcss](https://tailwindcss.com/)
